Установка:
1. Установите Python 3.11 или новее.
2. Скачайте архив со скриптом.
3. Распакуйте архив в удобную папку.
4. Откройте терминал в папке проекта.
5. Выполните команду:
pip install -r requirements.txt
6. После установки зависимостей запустите:
python main.py

Частые ошибки:

ModuleNotFoundError
Решение:
Обычно это значит, что не установлены зависимости. Выполните:
pip install -r requirements.txt

python не является внутренней или внешней командой
Решение:
Установите Python и добавьте его в PATH.

Permission denied
Решение:
Запустите терминал или скрипт от имени администратора.
